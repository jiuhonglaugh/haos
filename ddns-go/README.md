# ddns-go for Home Assistant

![ddns-go logo](https://raw.githubusercontent.com/jeessy2/ddns-go/master/web/static/logo.png)

一个轻量、简单、支持多平台的 DDNS（动态域名解析）工具，现已打包为 Home Assistant Add-on，方便一键部署！

> 基于官方 [ddns-go](https://github.com/jeessy2/ddns-go) 项目。

---

## 🧩 功能特性

- 支持阿里云、腾讯云、Cloudflare、华为云、DNSPod 等主流 DNS 服务商
- Web 图形界面配置（首次访问自动引导）
- 自动检测公网 IP 并更新 DNS 记录
- 轻量级，资源占用低
- 与 Home Assistant 深度集成，无需额外设备

---
